# Student Performance Analysis

## Internship Project – SoftGrowTech

This project was done during my Data Science internship at SoftGrowTech.
The main goal of this project is to analyze student data and understand what factors affect exam performance.

---

## About the Project

In this project, I worked with a student dataset and analyzed how different factors like study time, attendance, teacher quality, and activities impact exam scores.

I used Python to clean the data and create charts to find useful insights.

---

## Tools Used

* Python
* Pandas
* Matplotlib

---

## Dataset

* File: `Student_Performance.csv`
* Includes data like:

  * Exam Score
  * Study Hours
  * Attendance
  * Teacher Quality
  * School Type
  * Extracurricular Activities

---

## What I Did

### 1. Study Hours vs Exam Score

I divided students into levels (Low, Medium, High, Very High) based on study hours and compared their scores.

👉 Result: More study hours = better scores

---

### 2. Attendance vs Score

I checked how attendance affects marks.

👉 Result: Students with good attendance scored higher

---

### 3. Extracurricular Activities

I compared students who participate in activities and those who don’t.

👉 Result: Balanced participation improves performance

---

### 4. School Type Analysis

I found how many students scored above 90 based on school type.

👉 Result: Some school types have more top performers

---

### 5. Teacher Quality vs Pass Percentage

I analyzed how teacher quality affects student results.

👉 Result: Better teachers → higher pass percentage

---

##  Key Insights

* Study time is very important
* Attendance directly affects marks
* Good teachers improve student success
* Activities help if balanced
